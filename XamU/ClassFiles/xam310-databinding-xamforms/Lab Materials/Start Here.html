Before you begin

This is a Xamarin.Forms class which includes projects for iOS, Android, and Windows. When working on the exercises, you can select the easiest supported platform to run your application on. Refer to the Setup Requirements on the Xamarin University website if you need help with your development environment.

Overview

This session introduces the data binding infrastructure that is built into Xamarin.Forms. You will learn how to take advantage of this feature in your Xamarin.Forms applications to reduce the amount of code necessary to present data in your screens.

Objectives

  1. Introduction to Data Binding
  2. Dealing with Incompatible Types

Lecture

XAM310 - Data Binding in Xamarin.Forms.

Exercises

The lab exercises utilize image resources for various characters from the Simpsons TV series and were obtained with permission from simpsoncrazy.com.

  1. Using Data Binding in a Xamarin.Forms Application
    Exercise
    This group exercise will introduce creating and using bindings in code-behind and XAML.
  2. Working with Two-Way Bindings
    Exercise
    This exercise will implement two-way binding for our UI and ensure the UI is updated when the underlying data is changed by implementing the INotifyPropertyChanged interface.
  3. Using Value Converters
    Exercise
    There isn't always a 1:1 mapping between our available data and the user interface that presents that data. Value Converters are the bridge which spans this difference. In this exercise, we will continue editing our contacts application and add support to data bind images to our resources.
