Overview

This course shows you how to use Resources to eliminate repeated XAML. You will remove hard-coded values for things like fonts and colors and define them as resources so you have a single copy that is easy to update. You will also consolidate groups of property values into Styles which you will apply to a control as a single unit. This makes it much easier to create a consistent UI across your app.

Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a physical device. Please see the setup requirements if you need guidance installing the Xamarin development environment.

Objectives

  1. Avoid duplicate XAML with Resources
  2. Create consistent UI with Styles
  3. Make your Resources and Styles available across your entire app
  4. Apply the user's Accessibility choices with built-in Styles
